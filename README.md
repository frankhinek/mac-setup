# mac setup
> macOS setup, powered by Ansible

## Compatibility

This Ansible playbook and the associated tasks have only been tested on macOS X
El Capitan.

## Roadmap

- [x] Install Homebrew
- [x] Install Common macOS Utilities
- [ ] Download dotfiles
- [ ] Install zsh
- [ ] Change default shell to zsh
- [ ] Install Docker for Mac
